Get Verified on TRUEX

TRUEX is a public betting record verification engine. Append-only. Lock-time enforced. No edits.

Who this is for:
Serious bettors who want a public, verifiable record — not a picks marketplace, not a feed, not a hype page.

What you get

  • A public verified profile page (shareable link)
  • Immutable performance stats (units, ROI, win rate, drawdowns, sport splits)
  • A TRUEX Verified badge you can embed anywhere
  • Raw analytics JSON for transparency

What TRUEX is not

  • No picks selling
  • No deleting losses
  • No backfilling old bets
  • No social feed / comments / ranking ladder

How verification works

  1. You submit bets in a CSV format
  2. Bets must be submitted before event start (lock-time enforced)
  3. Approved bets become part of a permanent public record
  4. Your profile updates on scheduled verification cycles

Submission checklist

  • ✅ Use UTC timestamps (ISO format like 2025-10-01T17:05:00Z)
  • ✅ Include odds and stake for every bet
  • ✅ One row per bet (no multi-legs unless labeled clearly)
  • ✅ Submit before event start time
  • ✅ Username must match exactly across submissions

CSV format

username,sport,event,market,selection,odds,stake,timestamp_utc

Example:

sampleuser,NFL,SEA vs NE,Spread,NE -5,-110,1.0,2025-10-01T17:05:00Z
