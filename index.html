What TRUEX does

TRUEX makes verified performance normal. It is not a social app, not a picks marketplace, and not a feed.

Prevents editing: bets can be added, not edited or deleted

Stops backfills: late submissions are rejected by lock-time rules

Publishes proof: raw analytics + verification metadata are public

Creates shareable profiles: one link that anyone can verify

Core principle
If it can be edited, it can’t be trusted.
